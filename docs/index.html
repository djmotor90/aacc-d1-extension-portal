Common Issues

Extension doesn't work on D1 pages

Make sure you're on a Student Profile page in DestinyOne. The extension only activates on URLs containing 'studentProfile'. Check the extension popup for status information.

Custom Fields section not moving

Try refreshing the page or clicking "Move Custom Fields" in the extension popup. The extension looks for specific elements that may load dynamically.

How to update the extension

Download the latest version from this portal, extract it, and replace your existing extension folder. You may need to reload the extension in Chrome.
